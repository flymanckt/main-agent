# 观察池管理规则

## 一、纳入标准
标的进入观察池，必须至少满足 3 条：
1. 有明确催化（公告/业绩/政策/主题/板块）
2. 有清晰量价结构
3. 有明确触发条件
4. 有明确失效条件
5. 有可接受的风险收益比

## 二、优先级划分
### A 级
- 催化强
- 板块共振
- 量价结构成熟
- 可随时进入重点跟踪

### B 级
- 逻辑存在，但触发条件未成熟
- 适合继续观察，不急于出手

### C 级
- 仅有初步逻辑或消息刺激
- 暂不作为重点提示对象

## 三、移出条件
出现任一项，考虑移出观察池：
1. 核心逻辑证伪
2. 跌破关键结构且无修复
3. 板块退潮明显
4. 催化兑现后无承接
5. 连续多日不再具备赔率

## 四、升级规则
- C → B：有新增确认信息
- B → A：满足触发条件，且信号评分 >= 7
- A → 持仓级跟踪：实际建仓后

## 五、维护节奏
- 每日收盘后复核一次
- 周末做一次清理
- 不允许观察池无限增长，默认控制在 5-15 只内
