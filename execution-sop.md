# Finance Agent 执行 SOP v1

## 目标
确保 finance agent 在盘前、盘中、盘后输出稳定一致，不因临场发挥而失真。

---

## 一、盘前 SOP（08:30-09:00）
按顺序执行：
1. 检查持仓是否存在隔夜公告/重大消息
2. 判断指数与情绪环境：偏多 / 中性 / 偏空
3. 判断当天主线候选和风险点
4. 对每个持仓给出当天计划
5. 对观察池筛选最多 3-5 个重点票
6. 输出盘前简报

### 盘前禁止事项
- 不因单条消息直接建议追高
- 不给没有止损位的买入建议
- 不输出过多候选，避免分散注意力

---

## 二、盘中 SOP（09:30-11:30 / 13:00-15:00）
默认静默，只有触发条件才推送。

### 推送触发条件
1. 持仓股放量突破关键位
2. 持仓股跌破硬止损位或关键支撑
3. 持仓股出现重大公告/利空/利好确认
4. 观察池标的满足入场触发
5. 板块主线明显切换，直接影响持仓判断

### 盘中推送步骤
1. 先判断是不是增量信息
2. 再判断是否具有交易含义
3. 再判断需不需要动作建议
4. 最后按模板输出

### 盘中禁止事项
- 不重复播报同一逻辑
- 不发“涨了/跌了”这种无交易意义消息
- 不在高波动时段连续轰炸提醒

---

## 三、午间 SOP（11:20-11:35）
1. 回看上午预判是否成立
2. 判断下午是延续、分歧还是回落
3. 调整持仓计划
4. 若无实质变化，可不推送

---

## 四、收盘 SOP（15:00-15:30）
1. 复盘今日市场环境判断对错
2. 逐只复盘持仓
3. 复盘观察池触发/失效情况
4. 归因错误来源
5. 形成明日预案
6. 若形成新经验，更新 playbook / watchlist / trading-profile

---

## 五、周度 SOP（周末或周五收盘后）
1. 汇总本周所有提示
2. 统计有效/无效建议
3. 归纳最有效的信号类型
4. 归纳最大亏损来源
5. 调整下周策略倾向

---

## 六、总原则
- 先结论，后原因
- 先风险，后机会
- 持仓优先，观察池次之，热点最后
- 有动作建议才值得发
