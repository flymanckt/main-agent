## 一、量化策略体系

### 1. 网格交易策略

**核心原理：**
在震荡行情中，将价格区间划分为若干等宽网格，价格每下跌一格买入，上涨一格卖出，积小胜为大胜。

**关键参数（A股适用）：**
- 趋势判断标准：`MA20（20日均线）+ MACD` 双指标确认方向
  - MA20向上 + MACD金叉 → 上升趋势（暂停做空网格）
  - MA20向下 + MACD死叉 → 下降趋势（暂停做多网格）
  - 两者矛盾 → 震荡趋势（网格最佳环境）
- 网格数量：A股波动特性建议 8-20 格
- 止盈线：单格收益 1-3%
- 仓位管理：不一把梭，每次网格等额买入

**A股注意事项：**
- A股有涨跌停限制，网格间距需≥10%
- 配合ETF效果更佳（流动性好、无个股黑天鹅）
- 熊市/震荡市效果最好，单边牛市会卖飞

---

### 2. 双均线策略

**核心原理：**
短期均线上穿长期均线（金叉）买入，下穿（死叉）卖出。

**Python实现要点：**
```python
# 经典双均线
MA5 = close.rolling(5).mean()
MA20 = close.rolling(20).mean()
signal = (MA5 > MA20) & (MA5.shift(1) <= MA20.shift(1))  # 金叉
```

**参数优化方向：**
- 短期均线：常用 5/10/12 日
- 长期均线：常用 20/30/60 日
- 需根据股性（波动大小）调整

**缺陷：**
- 滞后性强，会买在山顶、卖在谷底
- 单边行情有效，震荡市频繁假信号

### 3. RSI动量策略

**RSI核心用法：**
- RSI > 70：超买区，回调风险，分批止盈
- RSI < 30：超卖区，可能反弹，可小仓试探
- RSI 50：多空平衡线

**RSI背离（高价值信号）：**
- **顶背离**：股价创出新高，RSI高点却比前一次低 → 趋势动能衰竭，强烈卖出信号
- **底背离**：股价创出新低，RSI低点却比前一次高 → 趋势见底，潜在买入信号

**结合RSI的改进均线策略：**
1. 只在MA多头排列时参考RSI买入信号
2. RSI<30时出现金叉 → 买入；RSI>70时出现死叉 → 卖出

---

### 4. MACD背离策略

**MACD三要素：**
- DIF线（快线）= EMA12 - EMA26
- DEA线（慢线）= DIF的EMA9
- MACD柱 = (DIF - DEA) × 2

**背离判断：**
- **顶背离**：价格创新高，DIF/柱没有创新高 → 看跌
- **底背离**：价格创新低，DIF/柱没有创新低 → 看涨
- MACD背离比RSI背离更可靠（过滤了部分噪音）

**量化识别MACD背离的核心逻辑：**
```
价格创新高 BUT DIF峰值 < 前一峰值 → 顶背离
价格创新低 BUT DIF谷值 > 前一谷值 → 底背离
```

**MACD + MA20 双确认：**
- 趋势方向由MA20决定
- 买入时机由MACD金叉/底背离确认
- 两者共振信号最强

## 二、技术分析进阶

### 1. 缠论（缠中说禅）

**核心框架：**
缠论是本土技术分析体系，以纯几何定义将股价波动分类，追求"不测而测"。

**层级结构：**
```
K线 → 分型（顶分型/底分型）→ 笔 → 线段 → 中枢
```

**核心概念：**
- **分型**：三根K线组合，中间一根最高/最低
- **笔**：至少5根不包含包含关系的K线，上升/下降
- **线段**：三笔重叠构成，走势的基本单元
- **中枢**：至少三段重叠区域，多空博弈的核心区间
- **背驰/背离**：走势的内在动能衰竭，用MACD/RSI辅助判断

**三类买卖点：**
- **一买**：下跌趋势背驰后的底点（最安全但最难判断）
- **二买**：回踩前一低点不破的买入点
- **三买**：突破中枢后的回踩确认点

**实战价值：**
- 提供了唯一解的递归体系
- 适合30分钟以上周期，不适合分时高频
- 难点：K线包含处理 + 递归级别判断

### 2. 波浪理论（艾略特）

**基本结构：8浪循环**
```
上升5浪（推动浪）：
  1浪 → 2浪调整 → 3浪（通常最长）→ 4浪调整 → 5浪

下跌3浪（调整浪）：
  A浪 → B浪反弹 → C浪（通常等于A浪长度）
```

**各浪特征：**
- **1浪**：起始，上涨温和，常被误认为是反弹
- **2浪**：回调深（常50-78.6%回撤1浪），成交量减少
- **3浪**：爆发力最强，量能最大，趋势最明确
- **4浪**：调整（常38.2%回撤3浪），形态复杂，不能重叠1浪顶部
- **5浪**：量能可能减弱，常出现背离

**斐波那契比率应用：**
- 2浪回调：常回撤1浪的50%、61.8%、78.6%
- 4浪回调：常回撤3浪的38.2%
- 3浪涨幅：通常为1浪的161.8%
- 5浪涨幅：通常为1浪至3浪总涨幅的61.8%

**A股实战注意：**
- 波浪有主观性，不同人划分可能不同
- 需结合成交量、均线、MACD共振确认
- 结构性行情（2019年以来）3浪主升段更明显

---

### 3. 支撑与阻力

**支撑位：** 价格下跌时遇到买盘支撑的区域（"地板"）

**阻力位：** 价格上涨时遇到卖压的区域（"天花板"）

**关键支撑/阻力类型：**
1. **前期高低点**：近期的显著高/低点
2. **均线支撑**：MA5/MA10/MA20/MA60
3. **整数关口**：如3000、3500点
4. **跳空缺口**：A股常见的缺口回补
5. **成交密集区**：横盘整理区域

**趋势线：**
- 上升趋势线：连接两个以上低点
- 下降趋势线：连接两个以上高点
- 角度越陡，可靠性越强（但越容易突破）

---

### 4. 假突破识别

**假突破特征（诱多/诱空陷阱）：**
1. **成交量**：真突破需放量（>20日均量30%以上），假突破量能萎缩
2. **K线形态**：突破后收上影线/阴线 → 可疑
3. **幅度**：突破幅度<3%且快速回落 → 大概率假突破
4. **时间**：A股收盘前30分钟判断更准
5. **回踩确认**：突破后回踩不破原阻力 → 真突破

**应对策略：**
- 等待"回踩确认"再入场（突破后2-3天不跌回）
- 轻仓试探，确认后加仓
- 设置止损在突破点下方3-5%

## 三、量化回测框架

### 工具栈（A股）

| 工具 | 用途 |
|------|------|
| **AKShare** | 免费开源，A股数据全覆盖 |
| **Backtrader** | 专业回测框架，支持多股/多策略 |
| **PyBroker** | 支持机器学习驱动策略 |
| **QuantStats** | 策略绩效可视化分析 |

### 关键回测指标

- **年化收益率（Annual Return）**：策略收益 vs 基准
- **最大回撤（Max Drawdown）**：从峰值到谷底最大跌幅
- **夏普比率（Sharpe Ratio）**：风险调整后收益，>1为合格
- **胜率（Win Rate）**：盈利交易/总交易
- **盈亏比（Profit Factor）**：总盈利/总亏损，>1.5较好
- **卡尔马比率（Calmar）**：年化收益/最大回撤

### 避免过拟合
- 样本内测试 ≠ 样本外真实表现
- 参数不要过度优化（均线参数来回调试最常见陷阱）
- 用最近1-2年数据做样本外验证

## 四、策略体系整合

**我的操盘框架（更新版）：**

```
趋势判断：
  MA5 > MA10 → 多头 | MA5 < MA10 → 空头

入场信号（多指标共振）：
  ✅ MA多头排列 + RSI底背离/超卖 + 放量 → 买入
  ✅ 缠论底分型 + 放量 → 买入
  ✅ 波浪2/4浪回调结束 + MACD金叉 → 买入

止损纪律：
  - 亏损<7%：可持有观察
  - 亏损7-10%：必须决策（减仓或止损）
  - 亏损>10%：无条件止损

出场信号：
  ⚠️ RSI顶背离 + MA死叉 → 分批止盈
  ⚠️ MACD顶背离 → 离场
  ⚠️ 波浪5浪走完 + 放量滞涨 → 离场
  ⚠️ 假突破确认 → 止损/止盈

网格补充：
  震荡行情（MA20走平 + MACD零轴附近）：启用网格收割波动
  单边行情：网格暂停，趋势策略为主
```
