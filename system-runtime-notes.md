# Finance Agent 运行说明

## 当前能力边界
当前 finance agent 已具备：
1. 持仓优先的输出框架
2. 盘前/盘中/盘后固定模板
3. 信号评分卡
4. 观察池治理规则
5. 复盘台账框架
6. 结构化持仓与观察池文件

## 仍待增强
1. 实时行情接入
2. 公告/新闻自动抓取
3. 板块热度与资金流数据
4. 自动写入复盘台账
5. 周度统计自动化

## 当前最优使用方式
在无外部实时数据接入前，finance agent 应：
- 优先分析持仓与已知观察池
- 用严格模板减少主观漂移
- 用评分卡过滤噪音机会
- 用复盘台账持续迭代规则

## 优化方向
后续若接入数据源，应优先接：
1. A股行情
2. 交易所公告
3. 财经快讯
4. 板块强弱/成交额

## 结论
目前已达到“高质量人工监督型交易助理”配置。
距离“半自动策略系统”，还差数据源和自动执行层。
