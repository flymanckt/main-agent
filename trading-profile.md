# Kent 交易档案 v1
> 用于 finance agent 的个性化决策基线

## 一、目标定位
- 目标：通过更高质量的信息、纪律与复盘，提升 A 股交易决策质量
- 风格倾向：短线 + 波段 + 事件驱动混合
- 核心诉求：明确买卖提示，不要泛泛而谈

## 二、风险偏好（当前按保守-中性预设）
- 可接受技术试错，但不接受无纪律扛亏
- 默认重视止损和失效条件
- 对重大操作保持谨慎
- 情绪退潮期优先防守

## 三、仓位原则（初版）
- 持仓股优先于新开仓
- 新机会默认先轻仓试错，不一次打满
- 没有清晰触发和止损，不建议开仓
- 单票仓位上限：待后续确认

## 四、偏好机会类型
1. 有明确事件催化的个股
2. 趋势结构清晰的强势股
3. 有板块共振的机会
4. 风险收益比较好的低吸 / 回踩确认机会

## 五、回避类型
1. 纯消息传闻票
2. 连板高位末端接力
3. 无量上涨
4. 跌破关键位后幻想修复
5. 没有止损计划的交易

## 六、待补全信息
- 单票最大仓位
- 总仓位容忍区间
- 可接受单笔亏损比例
- 更偏短线还是波段
- 是否接受 ETF 轮动增强
