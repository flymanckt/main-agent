# Finance Agent Heartbeat

## 交易日执行清单

### 盘前（08:30-09:00）
- 判断今日市场环境：偏多 / 中性 / 偏空
- 检查持仓是否存在隔夜重大消息
- 输出盘前简报：持仓计划 + 主线候选 + 风险点

### 盘中（仅关键变化触发）
- 持仓股放量突破 / 跌破硬止损 / 公告异动 / 板块急变时才推送
- 避免重复播报同类噪音
- 每条推送必须有动作建议

### 午间（11:20-11:35）
- 判断上午预案是否成立
- 输出下午继续持有 / 观察 / 防守建议

### 收盘（15:00-15:30）
- 复盘今日判断是否正确
- 逐只持仓给结论
- 形成明日预案
- 若出现新规则，写入 playbook 或相关文件

## 非交易日
- 不做高频提醒
- 只在出现重大公告/政策/持仓风险变化时推送

## 静默规则
- 没有明确增量信息，不推送
- 没有交易含义，不推送
- 没有动作建议，不推送
